Shape: Spiral

Spiral Function with Parameters


import turtle

def spiral(sides, turn, color, width):
dizzy = turtle.Turtle()
dizzy.color(color)
dizzy.width(width)
dizzy.speed(5)

for n in range(sides):
dizzy.forward(n)
dizzy.right(turn)
